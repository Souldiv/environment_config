## Herbstluftwm

### Install using

```bash
apt install Herbstluftwm
```

### Configuration

```bash
touch ~/.config/herbstluftwm/autostart
```

### Synaptic drivers (for tap to click)

[See this](https://unix.stackexchange.com/questions/337008/activate-tap-to-click-on-touchpad)

### Adding $MOD1-p dmenu

```bash
apt install dmenu
```
